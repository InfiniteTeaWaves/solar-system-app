App details

This project is a Vanilla JS application which uses Three.js for the 3D simulation and Vite.js for tooling.
All important planet parameters are stored in a json file, such as the rotation angle, size or orbit distance, which allows to quickly adjust all relevant simulation parameters. Part of the CSS coding was generated with AI.
You can check out the full coding on Github.

Textures & models

The planet related textured are taken from solarsystemscope.
The textures are originally based on NASA elevation and imagery data. The space station is a model from the
3D Nasa Resources library.
